Module: EventMaster

Creates, removes and keeps track of listeners

( Static Functions )

Events will be removed every page load by PageLoader However, PageLoader is not required to use this. You can/should load AddEventListener into your individual view modules to build them independently.

You can use the others too, but AddEventListener was specifically made for view modules
