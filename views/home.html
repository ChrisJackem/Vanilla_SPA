A Vanilla SPA framework

No additives. No preservatives. 100% all natural JavaScript.


This Single Page Application framework relies on no other framework, plugin, Library, or anything else.


Developed in a few days and as simple as possible, you can scale this to a bigger project, use it as-is for a simple website template, or just check out my code and maybe get some ideas on how to approach your own SPA project.
